# Gestor

**Gestor** is a simple web application for publishing content build in Python.

## Features

- Tags
- Login
- Search
- Comments
- Markdown
- Multiple languages

## Requirements

- Python 2.7
- PIP

## Install

Install Dependencies.

```
pip install -r requirements.txt
```

Run Gestor

```
python main.py 127.0.0.1:8000
```
